# 需求背景

当前 `mobile/` 已经是独立的 Vue 3 H5 MPA 框架，但只有 `has-doc` 组件库示例应用，还没有面向真实业务的 `has-web` 入口。

你希望设计 `mobile/has-web` 的：

- 登录注册
- 首页
- 个人中心
- 接口对接方式

并且要和现有知识库中的 `helloagents/wiki/mobile.md`、`helloagents/wiki/c-end-integration.md` 对齐。

## 目标

1. 让 `mobile/has-web` 成为真实业务入口，而不是示例页。
2. 登录注册、首页、个人中心三块统一接后端认证与资料接口。
3. 页面结构、接口封装、状态管理、上传能力都能为后续开发复用。
4. 文档先行，后续开发时可以直接按文档落地。

## 非目标

- 不在本次方案里实现具体页面代码
- 不重构后端账号域
- 不引入新的认证协议
